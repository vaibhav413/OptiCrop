# 🌱 OptiCrop

### Intelligent Crop Recommendation System using Machine Learning

OptiCrop is a Flask-based web application that uses Machine Learning to recommend the most suitable crop based on soil nutrient values and environmental conditions.

---

## 🛠 Tech Stack

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Joblib
