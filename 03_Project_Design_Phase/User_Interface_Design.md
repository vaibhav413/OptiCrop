# User Interface Design

## Home Page

The home page of **OptiCrop** includes:

- Navigation Bar
- Project Title
- Project Description
- Hero Image
- Crop Recommendation Form
- Example Input Values
- About Section
- Footer

---

## Input Fields

The user enters the following details:

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Soil pH
- Rainfall

---

## Buttons

- **Try Prediction**
- **Predict Crop**
- **Predict Again** (Result Page)

---

## Result Page

The result page displays:

- Recommended Crop
- Prediction Confidence Score
- Soil & Climate Summary
- Success Message

---

## Design Principles

- Responsive Design
- Simple Navigation
- User-Friendly Interface
- Mobile-Friendly Layout
- Clean and Modern Design
